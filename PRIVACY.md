# Privacy Policy for YT Speed+

**Last Updated**: December 11, 2025

## Introduction

YT Speed+ ("the Extension") is committed to protecting your privacy. This Privacy Policy explains our practices regarding data collection, usage, and storage.

## Data Collection

**YT Speed+ does NOT collect, transmit, or share any personal data.**

### What We Store Locally

The Extension only stores:
- **Playback speed preference** - Your selected video speed (e.g., 1.5×)

This data is:
- Stored locally in your browser using Chrome's sync storage
- Never transmitted to external servers
- Never shared with third parties
- Only accessible by the Extension
- Synced across your Chrome browsers if you're signed into Chrome (handled by Chrome, not us)

### What We Do NOT Collect

- ❌ Browsing history
- ❌ Personal information
- ❌ YouTube watch history
- ❌ Video preferences
- ❌ Click behavior
- ❌ Analytics or usage statistics
- ❌ Cookies or tracking identifiers
- ❌ IP addresses
- ❌ Device information

## Permissions Explained

The Extension requests the following permissions:

### 1. Storage Permission
**Purpose**: Save your playback speed preference locally
**Usage**: Stores a single value (your preferred speed) in Chrome's local storage
**Data Access**: No external access; data never leaves your device

### 2. Tabs Permission
**Purpose**: Detect if the current tab is YouTube.com
**Usage**: Read the URL of the active tab to enable/disable controls appropriately
**Data Access**: Only checks if URL contains "youtube.com"; no other tab data is accessed

### 3. Host Permissions (youtube.com)
**Purpose**: Apply playback speed to YouTube videos
**Usage**: Inject content script to control video playback rate
**Data Access**: Only interacts with video player elements; no content or user data is accessed

## Third-Party Services

YT Speed+ does NOT use:
- Analytics services (e.g., Google Analytics)
- Crash reporting tools
- Advertising networks
- External APIs or servers
- Content Delivery Networks (CDNs) - except Google Fonts for UI typography

## Data Retention

- Speed preferences are stored indefinitely until you uninstall the Extension
- Uninstalling the Extension permanently deletes all stored data
- You can manually reset settings using the "Reset" button in the popup

## Children's Privacy

YT Speed+ does not knowingly collect any information from anyone, including children under 13.

## Changes to YouTube Content

YT Speed+ only modifies:
- Video playback speed (the technical rate at which videos play)

It does NOT:
- Modify video content
- Inject ads
- Track viewing behavior
- Interfere with YouTube's functionality beyond playback speed

## Your Rights

You have the right to:
- Remove the Extension at any time
- Clear stored preferences via the reset button
- Review the source code (available on request or via Chrome Web Store)

## Data Security

- All data is stored using Chrome's secure storage APIs
- No data transmission occurs
- No external servers are contacted
- Code is static and contains no dynamic script loading

## Updates to This Policy

We may update this Privacy Policy occasionally. Changes will be reflected by updating the "Last Updated" date. Continued use of the Extension after changes indicates acceptance of the updated policy.
